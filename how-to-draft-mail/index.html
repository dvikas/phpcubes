How to Draft Email for Google Groups

  • Mail Subject (Mandatory)
  • Body(Mandatory)
  • Add Reference
  • Attachment
  • Don't Do that

Post My Query »

  • Mail Subject (Mandatory)

    1. Mail without SUBJECT will never considered.
    2. Subject Should be explanatory.
    3. It should be not short like question, query ,Not working etc.
    4. It should give overview of the whole scenario.

  • Body(Mandatory)

    1. Please provide step by step explanation of your problem.
    2. If your code is more than 25 lines then please paste your code on http://pastebin.com or attach it as file.
    3. For javascript queries you should try to link it on http://jsfiddle.net/ eg http://jsfiddle.net/NnXex/1/ as it will save the time of viewers bcoz some time we dont have environment to test your code.
    4. For php you can use http://phpfiddle.org/ to show your php code.

  • Add Reference (Show your effort what you did to resolve your query)

    1. List your search result you got from google.
    2. Don't let the query to understood by respondent, if user is not able to understand your query then you will not get quick response.
    3. It help other viewers to get to the query faster as the the replier don't have to do extra effort.

  • Attachment

    1. Must attach a file, Image or screenshot of window if required.

  • Don't Do that

    1. Never spoil any one thread by asking your query in a thread, Always initiate a new Thread for your query.
    2. Try to avoid saying "Thanks" if someone share information/link.

Finally : Your thanks is valuable if suggestion provided by anyone works.
